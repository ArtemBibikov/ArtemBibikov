**Проект:** масштабируемая backend-платформа сбора клиентских отзывов  
(ролевая сегментация, автоматическая сортировка, маршрутизация уведомлений, интеграции с 15+ внутренними сервисами)

**Ключевые достижения:**

- Довёл платформу до **production‑ready** — отзывы стали обрабатываться мгновенно вместо ручной сортировки
- Настроил централизованное логирование (Sentry + логи) — время диагностики инцидентов сократилось в **2–3 раза**
- Оптимизировал API с асинхронной архитектурой — **p95 latency снизился с 500+ мс до <100 мс**
- Автоматизировал уведомления через очереди задач — команды тратят на обработку отзывов **минуты вместо часов**
- Реализовал **ролевую сегментацию (RBAC)** — каждый специалист получает только свои отзывы, обработка ускорилась в **3 раза**
